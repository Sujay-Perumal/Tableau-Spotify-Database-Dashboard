## Project Overview

This project is a dynamic, interactive dashboard built in Tableau, focused on analyzing key metrics and trends within a **Spotify music dataset**.

The goal was to transform raw data into actionable insights related to music attributes (e.g., danceability, energy, tempo), artist performance, and generational trends in music consumption.

## Skills Demonstrated

This dashboard showcases expertise in end-to-end data visualization and **Business Intelligence (BI)** development using Tableau:

| Skill Focus | Description |
| :--- | :--- |
| **Dashboard Design** | Creating a clean, effective, and cohesive dashboard layout that guides users through the data story. |
| **Calculated Fields** | Creating custom variables and metrics (e.g., categorization of tempo, normalizing scores) using Tableau's formulas. |
| **Data Cleaning & Prep** | Connecting to, structuring, and preparing the raw data source for efficient visualization. |
| **Interactivity** | Implementing action filters, highlight actions, and drill-down functionality to enable user exploration. |
| **Parameters & Controls** | Utilizing parameters for dynamic filtering, custom sorting, and "What-If" scenario analysis (e.g., selecting a variable threshold). |
| **Advanced Visualization** | Employing specialized chart types, such as scatter plots for correlation analysis, distribution charts, and custom hierarchies. |
